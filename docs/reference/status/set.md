# status.set

Устанавливает новый статус для текущего пользователя или сообщества.

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя.</span>
</div>

## Вызов метода

```csharp
int result = await api.Status.SetAsync(text: "Слушаю отличную музыку!", groupId: 0);
```

</div>

<div class="vk-card">

# Параметры

| Параметр / Тип | Описание |
| :--- | :--- |
| **text** <br> `string` | Текст нового статуса. <br> <span style="color: var(--vp-c-text-3)">строка, обязательный параметр</span> |
| **groupId** <br> `int` | Идентификатор сообщества, если статус устанавливается сообществу. <br> <span style="color: var(--vp-c-text-3)">целое число, по умолчанию: 0 (установить статус пользователю)</span> |

</div>

<div class="vk-card">

# Результат

Возвращает `1` в случае успешного изменения статуса.

</div>
