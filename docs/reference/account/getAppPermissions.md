# account.getAppPermissions

Возвращает права доступа текущего приложения (битовую маску) для авторизованного пользователя.

<div class="vk-info">
  <span>💡 Для вызова этого метода не требуется авторизация пользователя (токен необязателен).</span>
</div>

# Вызов метода

```csharp
int permissions = await api.Account.GetAppPermissionsAsync();
```

</div>

<div class="vk-card">

# Параметры

Метод не принимает параметров запроса.

</div>

<div class="vk-card">

# Результат

Возвращает число типа `int` — битовую маску прав доступа (например: `1` — уведомления, `2` — друзья, `4` — фото, `1024` — заметки, `8192` — аудио).

</div>
