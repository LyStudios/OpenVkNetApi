# account.setOnline

Устанавливает статус текущего пользователя «онлайн».

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя. Статус «онлайн» автоматически спадает через 15 минут неактивности.</span>
</div>

# Вызов метода

```csharp
int result = await api.Account.SetOnlineAsync();
```

</div>

<div class="vk-card">

# Параметры

Метод не принимает параметров запроса.

</div>

<div class="vk-card">

# Результат

Возвращает `1` в случае успешного выполнения.

</div>
