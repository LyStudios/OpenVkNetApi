# account.setOffline

Устанавливает статус текущего пользователя «оффлайн».

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя.</span>
</div>

# Вызов метода

```csharp
int result = await api.Account.SetOfflineAsync();
```

</div>

<div class="vk-card">

# Параметры

Метод не принимает параметров запроса.

</div>

<div class="vk-card">

# Результат

Возвращает `1` в случае успешного выполнения.

</div>
