# account.ban

Добавляет пользователя в черный список текущего аккаунта (блокирует доступ).

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя. Заблокированный пользователь не сможет писать вам сообщения и просматривать вашу страницу.</span>
</div>

# Вызов метода

```csharp
int result = await api.Account.BanAsync(ownerId: 1);
```

</div>

<div class="vk-card">

# Параметры

| Параметр / Тип | Описание |
| :--- | :--- |
| **ownerId** <br> `int` | Идентификатор пользователя, которого необходимо заблокировать. <br> <span style="color: var(--vp-c-text-3)">целое число, обязательный параметр</span> |

</div>

<div class="vk-card">

# Результат

Возвращает `1` в случае успешной блокировки.

</div>
