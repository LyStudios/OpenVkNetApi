# notifications.markAsViewed

Помечает все уведомления текущего пользователя как прочитанные/просмотренные.

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя с правами доступа к уведомлениям.</span>
</div>

## Вызов метода

```csharp
int result = await api.Notifications.MarkAsViewedAsync();
```

</div>

<div class="vk-card">

# Параметры

Метод не принимает параметров.

</div>

<div class="vk-card">

# Результат

Возвращает `1` в случае успешного выполнения.

</div>
