# Объект FriendStatus (Статус дружбы)

Описывает статус дружбы между текущим авторизованным пользователем и другим пользователем.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **UserId** <br> `int` | Идентификатор целевого пользователя. |
| **Status** <br> `int` | Статус отношений: <br> `0` — не друзья, <br> `1` — отправлена исходящая заявка, <br> `2` — есть входящая заявка, <br> `3` — друзья. |

</div>
