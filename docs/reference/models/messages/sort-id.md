# Объект SortId (Идентификатор сортировки)

Вспомогательная модель, используемая при сортировке бесед. Содержит составной идентификатор из старшей и младшей части.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **MajorId** <br> `int` | Старшая часть идентификатора сортировки (определяет порядок бесед). |
| **MinorId** <br> `int` | Младшая часть идентификатора сортировки (уточняет порядок внутри группы). |

</div>
