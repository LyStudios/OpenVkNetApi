# Объект ConversationCanWrite (Возможность отправки сообщений)

Описывает, может ли текущий пользователь отправлять сообщения в данный диалог.

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Allowed** <br> `bool` | Разрешена ли отправка сообщений в этот диалог (`true` — да, `false` — заблокировано). |
