# Объект LongPollServerInfo (Параметры Long Poll сервера)

Содержит данные для подключения к Long Poll серверу для отслеживания событий в реальном времени.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Key** <br> `string` | Временный секретный ключ сессии Long Poll. |
| **Server** <br> `string` | Адрес сервера Long Poll (например, `https://example.com/nim72`). |
| **Ts** <br> `long` | Временная метка (номер последнего события) для старта прослушивания. |
| **Pts** <br> `long?` | Идентификатор последнего события для истории Long Poll (PTS). |

</div>
