# Объект Conversation (Беседа / Диалог)

Представляет собой диалог или групповую беседу в личных сообщениях.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Peer** <br> `ConversationPeer` | Информация о собеседнике или групповой беседе. |
| **LastMessageId** <br> `int` | Идентификатор последнего сообщения в беседе. |
| **InRead** <br> `int` | Идентификатор последнего прочитанного входящего сообщения. |
| **OutRead** <br> `int` | Идентификатор последнего прочитанного исходящего сообщения. |
| **UnreadCount** <br> `int` | Количество непрочитанных сообщений в диалоге. |
| **Important** <br> `bool` | Помечен ли диалог как важный. |
| **IsMarkedUnread** <br> `bool` | Помечен ли диалог пользователем как непрочитанный. |
| **CanWrite** <br> `ConversationCanWrite` | Информация о возможности отправки сообщений в этот диалог. |

</div>
