# Объект ConversationAndMessage (Беседа с последним сообщением)

Представляет единицу из списка бесед — обёртку, содержащую одновременно метаданные диалога и последнее сообщение в нём.

</div>

| Поле / Тип | Описание |

| **LastMessage** <br> [Message](/reference/models/messages/message) | Последнее сообщение в беседе. |
