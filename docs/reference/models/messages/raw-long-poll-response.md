# Объект RawLongPollResponse (Сырой ответ Long Poll)

Представляет "сырой" ответ от Long Poll сервера OpenVK. Используется для ручной обработки событий реального времени.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Ts** <br> `long` | Новый временной штамп (`ts`), который нужно передать в следующем Long Poll запросе. |
| **Updates** <br> `List<List<object>>` | Список полученных событий. Каждое событие — массив, где первый элемент — числовой код типа события. |
| **Failed** <br> `int?` | Код ошибки, если соединение устарело: `1` — устарел `ts`, `2` — устарел ключ, `3` — потеря соединения. |

</div>
