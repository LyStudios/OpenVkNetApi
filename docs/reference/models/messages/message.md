# Объект Message (Личное сообщение)

Представляет собой личное сообщение в диалоге или беседе.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор сообщения. |
| **UserId** <br> `int` | Идентификатор собеседника. |
| **PeerId** <br> `int` | Идентификатор назначения (диалога/беседы). |
| **FromId** <br> `int` | Идентификатор отправителя сообщения. |
| **Date** <br> `long` | Unix-время отправки сообщения. |
| **ReadState** <br> `int` | Состояние прочтения (`0` — непрочитано, `1` — прочитано). |
| **Out** <br> `int` | Направление сообщения (`0` — входящее, `1` — исходящее). |
| **Text** / **Body** <br> `string` | Текстовое содержимое сообщения. |
| **Attachments** <br> `List<Attachment>` | Список прикрепленных объектов (фотографии, аудио, видео, документы и др.). |
| **ForwardedMessages** <br> `List<Message>` | Список прикрепленных пересланных сообщений. |

</div>
