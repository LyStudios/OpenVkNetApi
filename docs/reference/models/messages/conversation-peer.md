# Объект ConversationPeer (Собеседник)

Представляет информацию о противоположном участнике беседы (пользователе, сообществе или групповом чате).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Идентификатор собеседника. |
| **Type** <br> `string` | Тип собеседника: `user` — пользователь, `group` — сообщество, `chat` — групповая беседа. |
| **LocalId** <br> `int` | Локальный идентификатор внутри чата (для групповых бесед). |

</div>
