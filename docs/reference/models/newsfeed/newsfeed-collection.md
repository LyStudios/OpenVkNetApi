# Объект NewsfeedCollection (Коллекция новостей)

Представляет собой список новостных записей с поддержкой постраничной пагинации.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Items** <br> `List<T>` | Список новостных элементов (обычно объектов `Post`). |
| **Count** <br> `int` | Общее количество элементов в выборке. |
| **NextFrom** <br> `string` | Строковый токен, необходимый для получения следующей страницы ленты новостей. |

</div>
