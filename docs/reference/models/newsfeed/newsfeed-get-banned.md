# Объект NewsfeedGetBanned (Черный список новостей)

Представляет собой список пользователей и сообществ, публикации которых скрыты из новостной ленты текущего пользователя.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Count** <br> `int?` | Общее количество скрытых источников. |
| **Groups** <br> `List<Group>` | Список скрытых сообществ (объекты `Group`). |
| **Members** <br> `List<int>` | Список идентификаторов скрытых пользователей. |
| **Profiles** <br> `List<User>` | Профили скрытых пользователей (объекты `User`). |

</div>
