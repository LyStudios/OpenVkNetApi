# Объект NotificationsCollection (Коллекция уведомлений)

Специализированная коллекция для получения списков уведомлений. Содержит отметку времени о последнем прочтении. Наследует свойства [Collection](/reference/models/collection).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **LastViewed** <br> `long?` | Unix-время последнего просмотра уведомлений пользователем. |
