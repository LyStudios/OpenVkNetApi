# Объект Video (Видеозапись)

Представляет собой видеозапись в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор видеозаписи. |
| **OwnerId** <br> `int` | Идентификатор владельца видеозаписи. |
| **UserId** <br> `int?` | Идентификатор пользователя, загрузившего видеозапись. |
| **Title** <br> `string` | Название видеозаписи. |
| **Description** <br> `string` | Описание видеозаписи. |
| **Duration** <br> `int` | Длительность видеозаписи в секундах. |
| **Views** <br> `int` | Количество просмотров видеозаписи. |
| **Comments** <br> `int` | Количество комментариев к видеозаписи. |
| **Image** <br> `List<PhotoSize>` | Список превью-изображений видео в различных разрешениях. |
| **CanComment** / **CanLike** / **CanRepost** / **CanSubscribe** / **CanAdd** <br> `int?` | Права доступа для текущего авторизованного пользователя. |

</div>
