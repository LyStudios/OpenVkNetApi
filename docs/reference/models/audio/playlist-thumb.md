# Объект PlaylistThumb (Обложка плейлиста)

Представляет метаданные и URL-ссылки различных размеров для обложки плейлиста/альбома аудиозаписей.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Width** <br> `int` | Ширина обложки в пикселях. |
| **Height** <br> `int` | Высота обложки в пикселях. |
| **Photo34** / **Photo68** / **Photo135** / **Photo270** / **Photo300** / **Photo600** / **Photo1200** <br> `string` | URL-адреса изображений обложки в соответствующих размерах (от 34x34 до 1200x1200 пикселей). |

</div>
