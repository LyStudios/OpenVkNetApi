# Объект AudioUser (Загрузивший пользователь)

Представляет собой информацию о пользователе инстанса, который загрузил данный аудиофайл.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор пользователя. |
| **Name** <br> `string` | Полное имя и фамилия пользователя. |
| **NameGen** <br> `string` | Полное имя пользователя в родительном падеже (например, *"Ивана Иванова"*). |
| **Photo** <br> `string` | URL-адрес профильного фото пользователя. |

</div>
