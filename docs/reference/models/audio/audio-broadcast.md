# Объект AudioBroadcast (Аудиовещание)

Представляет информацию об объекте (пользователе или сообществе), который ведет музыкальное вещание (трансляцию статуса).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Идентификатор вещающего пользователя или сообщества. |
| **Name** <br> `string` | Название (имя) вещающего объекта. |
| **ScreenName** <br> `string` | Короткое буквенное имя вещающего объекта. |
| **Type** <br> `string` | Тип объекта (`user` — пользователь, `group` — сообщество). |
| **IsClosed** <br> `bool` | Закрыто ли сообщество (для сообществ). |
| **IsMember** <br> `int` | Является ли текущий пользователь участником сообщества. |
| **Deactivated** <br> `bool?` | Был ли объект деактивирован/удален. |
| **StatusAudio** <br> [Audio](/reference/models/audio/audio) | Аудиозапись, транслируемая в данный момент. |

</div>
