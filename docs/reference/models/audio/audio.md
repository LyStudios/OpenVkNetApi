# Объект Audio (Аудиозапись)

Представляет собой аудиофайл (трек) в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **UniqueId** <br> `string` | Уникальный строковый идентификатор аудиозаписи. |
| **Id** / **AudioId** <br> `int?` | Числовой идентификатор аудиозаписи. |
| **Artist** <br> `string` | Имя исполнителя/группы. |
| **Title** <br> `string` | Название композиции. |
| **Duration** <br> `int?` | Длительность аудиозаписи в секундах. |
| **Url** <br> `string` | Прямая ссылка на воспроизведение аудиофайла. |
| **Listens** <br> `int?` | Количество прослушиваний трека. |
| **GenreStr** <br> `string` | Текстовое название жанра аудиозаписи. |
| **Album** <br> [Album](/reference/models/audio/album) | Объект аудиоальбома, к которому принадлежит трек (если есть). |

</div>
