# Объект AudioCollectionWithShuffle (Коллекция аудио с перемешиванием)

Представляет собой список аудиофайлов, который содержит специальный сид перемешивания (shuffle seed).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Items** <br> `List<Audio>` | Список аудиозаписей [Audio](/reference/models/audio/audio). |
| **Count** <br> `int` | Общее число элементов в коллекции. |
| **ShuffleSeed** <br> `string` | Временный сид для воспроизведения плейлиста в случайном порядке. |

</div>
