# Объект BroadcastItem (Трансляция аудио)

Представляет собой информацию об аудиозаписи, которая в данный момент транслируется в статус пользователя или сообщества.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Идентификатор трансляции. |
| **StatusAudio** <br> [Audio](/reference/models/audio/audio) | Аудиозапись, транслируемая в данный момент. |
