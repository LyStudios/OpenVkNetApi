# Объект Album (Аудиоальбом / Плейлист)

Представляет собой плейлист (аудиоальбом) в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор альбома. |
| **OwnerId** <br> `int` | Идентификатор создателя/владельца плейлиста. |
| **Title** <br> `string` | Название аудиоальбома. |
| **Description** <br> `string` | Описание аудиоальбома. |
| **Size** <br> `int` | Количество аудиозаписей в альбоме. |
| **Length** <br> `int` | Суммарная длительность всех аудиозаписей в альбоме (в секундах). |
| **Created** <br> `int` | Unix-время создания альбома. |
| **Modified** <br> `int?` | Unix-время последнего редактирования. |
| **CoverUrl** <br> `string` | URL-адрес обложки плейлиста. |
| **Accessible** / **Editable** / **Bookmarked** / **Searchable** <br> `bool` | Права и состояние плейлиста для текущего пользователя. |

</div>
