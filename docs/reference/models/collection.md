# Объект Collection (Универсальная коллекция)

Базовый контейнер для списков элементов, возвращаемых методами OpenVK API.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Count** <br> `int` | Общее количество элементов в выборке на сервере. |
| **Items** <br> `List<T>` | Список элементов на текущей странице результатов. |
