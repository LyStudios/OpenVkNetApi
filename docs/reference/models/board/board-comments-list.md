# Объект BoardCommentsList (Комментарии обсуждения)

Содержит список комментариев темы обсуждения, а также профили авторов (пользователей и сообществ).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Items** <br> `List<BoardComment>` | Список комментариев темы обсуждения. |
| **Profiles** <br> `List<User>` | Профили пользователей, являющихся авторами комментариев. |
| **Groups** <br> `List<Group>` | Профили сообществ, являющихся авторами комментариев. |

</div>
