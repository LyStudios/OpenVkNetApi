# Объект BoardTopic (Тема обсуждения)

Представляет тему в обсуждениях сообщества.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор темы обсуждения. |
| **CreatedBy** <br> `int` | Идентификатор пользователя, создавшего тему. |
| **Title** <br> `string` | Заголовок/название темы. |
| **Created** <br> `long` | Unix-время создания темы. |
| **Updated** <br> `long` | Unix-время последнего обновления (нового сообщения). |
| **UpdatedBy** <br> `long` | Идентификатор пользователя, оставившего последнее сообщение. |
| **Comments** <br> `int` | Общее количество комментариев (сообщений) в теме. |
| **IsClosed** <br> `bool` | Закрыта ли тема для новых сообщений. |
| **IsFixed** <br> `bool` | Закреплена ли тема вверху списка обсуждений. |
| **FirstComment** <br> `string` | Текст первого сообщения темы (превью). |
| **LastComment** <br> `string` | Текст последнего сообщения темы (превью). |

</div>
