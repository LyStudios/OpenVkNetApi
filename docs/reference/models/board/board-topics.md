# Объект BoardTopics (Список тем обсуждений)

Представляет собой список тем обсуждения в сообществе с дополнительными метаданными.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Items** <br> `List<BoardTopic>` | Список тем обсуждений сообщества. |
| **Count** <br> `int` | Общее количество тем в сообществе. |
| **CanAddTopics** <br> `bool?` | Может ли текущий пользователь создавать новые темы в данном сообществе. |

</div>
