# Объект NoteComment (Комментарий к заметке)

Представляет собой комментарий к заметке в OpenVK. Расширяет базовую модель [Comment](/reference/models/comments/comment) дополнительными свойствами.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **UserId** <br> `int` | Идентификатор автора комментария (`uid`). |
| **NoteId** <br> `int` | Идентификатор заметки (`nid`), к которой оставлен комментарий. |
| **OwnerId** <br> `int` | Идентификатор владельца заметки (`oid`). |

</div>
