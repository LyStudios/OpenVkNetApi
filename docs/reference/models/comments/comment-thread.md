# Объект CommentThread (Ветка комментариев)

Описывает ветку вложенных ответов на комментарий.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Count** <br> `int` | Количество ответов в ветке. |
| **Items** <br> `List<object>` | Список вложенных комментариев. |
| **CanPost** <br> `bool` | Может ли текущий пользователь оставлять ответы в этой ветке. |
| **ShowReplyButton** <br> `bool` | Нужно ли отображать кнопку ответа. |
| **GroupsCanPost** <br> `bool` | Могут ли сообщества оставлять ответы в этой ветке. |

</div>
