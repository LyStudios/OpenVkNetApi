# Объект WallComment (Комментарий к записи на стене)

Представляет комментарий к записи на стене. Расширяет базовую модель [Comment](/reference/models/comments/comment).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **CanEdit** <br> `bool` | Может ли текущий пользователь редактировать данный комментарий. |
| **CanDelete** <br> `bool` | Может ли текущий пользователь удалить данный комментарий. |
| **PostId** <br> `int` | Идентификатор записи на стене, к которой относится комментарий. |
| **OwnerId** <br> `int` | Идентификатор владельца стены. |
| **Thread** <br> [CommentThread](/reference/models/comments/comment-thread) | Информация о ветке вложенных ответов на этот комментарий. |
| **Likes** <br> `LikesInfo` | Информация об отметках "Мне нравится" на комментарии. |
| **IsFromPostAuthor** <br> `bool?` | Оставлен ли комментарий автором самой записи. |

</div>
