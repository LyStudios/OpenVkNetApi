# Объект Comment (Базовая модель комментария)

Базовая модель, представляющая комментарий в OpenVK. Большинство специфичных типов комментариев наследуют свойства этой модели.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор комментария. |
| **FromId** <br> `int` | Идентификатор автора комментария (пользователь или сообщество). |
| **Date** <br> `long` | Unix-время добавления комментария. |
| **Text** <br> `string` | Текстовое содержимое комментария. |
| **Attachments** <br> `List<Attachment>` | Список вложений к комментарию (фото, аудио, видео, документы и т.д.). |
| **ParentsStack** <br> `List<object>` | Стек идентификаторов родительских комментариев (для многоуровневых ответов). |
| **UserLikes** <br> `bool?` | Поставил ли текущий пользователь отметку "Мне нравится" на этот комментарий. |
| **CanLike** <br> `bool?` | Может ли текущий пользователь поставить отметку "Мне нравится" на этот комментарий. |

</div>
