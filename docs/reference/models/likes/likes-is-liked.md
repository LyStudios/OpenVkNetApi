# Объект LikesIsLiked (Статус отметки)

Содержит сведения о том, поставил ли пользователь лайк или сделал ли репост объекта.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Liked** <br> `bool` | Указывает, поставил ли текущий пользователь отметку "Мне нравится" на этот объект. |
| **Copied** <br> `bool` | Указывает, поделился ли текущий пользователь этим объектом на своей стене (сделал ли репост). |

</div>
