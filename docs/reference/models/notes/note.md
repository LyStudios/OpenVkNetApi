# Объект Note (Заметка)

Модель представляет собой заметку пользователя OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор заметки. |
| **OwnerId** <br> `int` | Идентификатор владельца заметки (пользователя, создавшего её). |
| **Title** <br> `string` | Заголовок заметки. |
| **Text** <br> `string` | Текстовое содержимое заметки (в формате HTML или wiki). |
| **Date** <br> `long` | Дата создания заметки в формате Unixtime. |
| **Comments** <br> `int` | Количество оставленных комментариев к этой заметке. |
| **ViewUrl** <br> `string` | Прямая ссылка для просмотра заметки в веб-интерфейсе. |

</div>
