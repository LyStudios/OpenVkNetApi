# Объект Geo (Географическая метка)

Содержит географические координаты и адрес места, прикрепленного к объекту.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Type** <br> `string` | Тип гео-метки (например, `"point"`). |
| **Coordinates** <br> `string` | Строковые координаты (широта, долгота) через запятую. |
| **Name** <br> `string` | Название географического места или адреса. |
| **Place** <br> `string` | Краткое описание места. |

</div>
