# Объект Auth (Авторизация)

Представляет собой результат успешной авторизации на сервере инстанса.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **AccessToken** <br> `string` | Токен доступа к API для совершения авторизованных запросов. |
| **ExpiresIn** <br> `int` | Время жизни токена доступа в секундах. |
| **UserId** <br> `int` | Уникальный числовой идентификатор авторизованного пользователя. |
| **IsStale** <br> `bool` | Указывает, устарел ли данный токен. |
| **Secret** <br> `string` | Секретный ключ авторизации (если возвращается сервером). |

</div>
