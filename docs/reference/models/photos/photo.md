# Объект Photo (Фотография)

Представляет собой фотографию в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор фотографии. |
| **AlbumId** <br> `int?` | Идентификатор альбома, которому принадлежит фотография. |
| **OwnerId** <br> `int` | Идентификатор владельца фотографии. |
| **UserId** <br> `int` | Идентификатор пользователя, загрузившего фотографию. |
| **Date** <br> `long` | Unix-время загрузки фотографии. |
| **Width** <br> `int?` | Ширина оригинального изображения в пикселях. |
| **Height** <br> `int?` | Высота оригинального изображения в пикселях. |
| **Photo75** / **Photo130** / **Photo604** / **Photo807** / **Photo1280** / **Photo2560** <br> `string` | URL-адреса копий фотографии различного разрешения. |
| **Url** <br> `string` | Прямая CDN-ссылка на фотографию. |
| **OriginalSource** <br> `string` | URL-адрес оригинального изображения максимального разрешения. |
| **Likes** <br> [PhotoLikes](/reference/models/photos/photo-likes) | Информация об отметках "Мне нравится" для этой фотографии. |
| **Comments** <br> [PhotoComments](/reference/models/photos/photo-comments) | Информация о комментариях к этой фотографии. |

</div>
