# Объект PhotosUploadServer (Сервер загрузки фотографий)

Описывает адрес сервера, полученный для последующей загрузки файлов фотографий в альбомы или на стену.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **UploadUrl** <br> `string` | URL-адрес, на который нужно отправить файл методом POST. |
| **AlbumId** <br> `int?` | Идентификатор альбома, для которого получен этот сервер загрузки. |
| **UserId** <br> `int` | Идентификатор пользователя, запросившего сервер загрузки. |
