# Объект PhotoComments (Информация о комментариях фотографии)

Содержит сведения о количестве комментариев к фотографии и правах на их оставление.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Count** <br> `int?` | Общее количество комментариев к данной фотографии. |
| **CanPost** <br> `bool?` | Может ли текущий пользователь комментировать эту фотографию. |
