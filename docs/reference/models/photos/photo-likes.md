# Объект PhotoLikes (Отметки "Мне нравится" фотографии)

Содержит сведения о количестве лайков на фотографии и отношении к ним текущего пользователя.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Likes** <br> `int?` | Общее количество пользователей, поставивших отметку "Мне нравится". |
| **UserLikes** <br> `bool?` | Поставил ли текущий пользователь лайк на эту фотографию. |
| **CanLike** <br> `bool?` | Может ли текущий пользователь поставить лайк на эту фотографию. |
| **CanPublish** <br> `bool?` | Разрешена ли публикация лайка. |

</div>
