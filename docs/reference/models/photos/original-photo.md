# Объект OriginalPhoto (Оригинал фотографии)

Содержит информацию о размерах и URL-адресе оригинального полноразмерного файла изображения.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Width** <br> `int` | Ширина оригинального изображения в пикселях. |
| **Height** <br> `int` | Высота оригинального изображения в пикселях. |
| **Url** <br> `string` | Прямая ссылка на скачивание оригинального файла изображения. |
| **Type** <br> `string` | Буквенный тип размера оригинального файла (обычно `"w"` или `"z"`). |

</div>
