# Объект PhotoSize (Размер копии фотографии)

Описывает размеры и URL-адрес для одной из копий изображения конкретного масштаба.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Type** <br> `string` | Буквенное обозначение размера копии (например, `s`, `m`, `x`, `y`, `z`, `w`). |
| **Url** <br> `string` | Прямая ссылка на файл изображения этого размера. |
| **Width** <br> `int?` | Ширина копии в пикселях. |
| **Height** <br> `int?` | Высота копии в пикселях. |
| **Crop** <br> `bool?` | Обрезано ли изображение по краям. |
| **WithPadding** <br> `int?` | Содержит ли изображение внутренние отступы (padding). |

</div>
