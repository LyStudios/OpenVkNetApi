# Объект Album (Фотоальбом)

Представляет собой альбом с фотографиями в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор альбома. |
| **ThumbId** <br> `int` | Идентификатор обложки альбома (фотографии). |
| **OwnerId** <br> `int` | Идентификатор владельца альбома. |
| **Title** <br> `string` | Название фотоальбома. |
| **Description** <br> `string` | Текстовое описание альбома. |
| **Created** <br> `long` | Unix-время создания альбома. |
| **Updated** <br> `long` | Unix-время последнего обновления альбома. |
| **Size** <br> `int` | Количество фотографий в альбоме. |
| **ThumbSrc** <br> `string` | URL-адрес обложки альбома. |

</div>
