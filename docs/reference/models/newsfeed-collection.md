# Объект NewsfeedCollection (Коллекция новостной ленты)

Специализированная расширенная коллекция результатов новостей, поддерживающая параметр `next_from` для пагинации. Наследует свойства [ExtendedCollection](/reference/models/extended-collection).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **NextFrom** <br> `string` | Временной маркер для получения следующей страницы новостной ленты. |
