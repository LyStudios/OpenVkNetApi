# Объект UserStatus (Статус)

Представляет текущий статус пользователя или сообщества в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Text** <br> `string` | Текстовая строка статуса. |
| **Audio** <br> [Audio](/reference/models/audio/audio) | Объект транслируемой аудиозаписи (если статус содержит трансляцию аудио). |

</div>
