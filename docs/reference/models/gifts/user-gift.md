# Объект UserGift (Полученный подарок)

Представляет собой информацию о подарке, полученном пользователем.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор полученного подарка. |
| **FromId** <br> `int` | Идентификатор отправителя подарка. |
| **Message** <br> `string` | Сопроводительное текстовое сообщение от отправителя. |
| **Date** <br> `long` | Unix-время отправки подарка. |
| **Privacy** <br> `int` | Уровень приватности подарка (виден всем / только получателю). |
| **Gift** <br> `Gift` | Сам объект подарка (содержит изображения различного размера). |

</div>
