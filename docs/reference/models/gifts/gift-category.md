# Объект GiftCategory (Категория подарков)

Представляет категорию подарков в каталоге.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор категории. |
| **Name** <br> `string` | Название категории подарков. |
| **Description** <br> `string` | Описание категории. |
| **Thumbnail** <br> `string` | URL-адрес обложки категории. |

</div>
