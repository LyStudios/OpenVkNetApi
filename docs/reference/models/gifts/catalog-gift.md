# Объект CatalogGift (Элемент каталога подарков)

Представляет собой информацию о подарке из каталога подарков.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Name** <br> `string` | Название подарка. |
| **Image** <br> `string` | URL-адрес изображения подарка. |
| **UsagesLeft** <br> `int` | Сколько раз данный подарок еще можно использовать (если есть ограничение). |
| **Price** <br> `int` | Стоимость подарка в голосах. |
| **IsFree** <br> `bool` | Является ли подарок бесплатным. |

</div>
