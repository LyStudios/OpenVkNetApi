# Объект GiftsSend (Результат отправки подарка)

Представляет результат выполнения операции отправки подарка другому пользователю.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Success** <br> `bool` | Указывает, была ли операция отправки подарка выполнена успешно. |
| **UserIds** <br> `int` | Идентификатор пользователя-получателя подарка. |
| **WithdrawVotes** <br> `int?` | Количество голосов, списанных за отправку подарка. |
| **Error** <br> `string` | Сообщение об ошибке (если операция завершилась неудачно). |

</div>
