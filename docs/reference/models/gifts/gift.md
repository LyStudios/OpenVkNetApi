# Объект Gift (Подарок)

Представляет собой медиа-объект подарка с изображениями разного разрешения.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор подарка. |
| **Thumb256** <br> `string` | URL-адрес превью-изображения размером 256x256 пикселей. |
| **Thumb96** <br> `string` | URL-адрес превью-изображения размером 96x96 пикселей. |
| **Thumb48** <br> `string` | URL-адрес превью-изображения размером 48x48 пикселей. |

</div>
