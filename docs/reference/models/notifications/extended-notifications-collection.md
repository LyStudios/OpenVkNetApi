# Объект ExtendedNotificationsCollection&lt;T&gt;

Представляет собой коллекцию уведомлений, которая включает в себя дополнительную информацию об упомянутых пользователях и сообществах, а также время последнего просмотра.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Count** <br> `int` | Общее количество уведомлений. |
| **Items** <br> `List<T>` | Список объектов уведомлений (например, [Notification](/reference/models/notifications/notification)) на текущей странице. |
| **LastViewed** <br> `long?` | Unix-время последнего просмотра уведомлений текущим пользователем. |
| **Profiles** <br> `List<ProfileInfo>` | Список профилей пользователей, упомянутых в уведомлениях. |
| **Groups** <br> `List<Group>` | Список сообществ, упомянутых в уведомлениях. |

</div>
