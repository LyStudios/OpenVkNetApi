# Объект Notification (Уведомление)

Представляет собой отдельное уведомление в системе OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `long` | Уникальный идентификатор уведомления. |
| **Type** <br> `string` | Тип уведомления (например, `"comment"`, `"like"`, `"friend_accepted"`). |
| **Date** <br> `long` | Unix-время создания уведомления. |
| **Parent** <br> `object` | Родительский объект, к которому относится уведомление (пост, фото, видео и т.д.). |
| **Feedback** <br> `object` | Объект с подробностями о событии (например, сам комментарий, информация о лайкнувшем и т.д.). |

</div>
