# Объект OvkAboutInstance

Содержит детальную информацию об инстансе OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Statistics** <br> [OvkAboutInstanceStatistics](/reference/models/ovk/ovk-about-instance-statistics) | Статистика по активности пользователей и контенту на инстансе. |
| **Administrators** <br> `Collection<User>` | Коллекция администраторов инстанса (объекты `User`). |
| **PopularGroups** <br> `Collection<Group>` | Коллекция популярных сообществ инстанса (объекты `Group`). |
| **Links** <br> `Collection<OvkAboutInstanceLink>` | Коллекция внешних ссылок, связанных с инстансом (например, GitHub проекта, правила использования и др.). |

</div>
