# Объект OvkTest

Представляет результат тестирования доступности API и авторизации.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Authorized** <br> `bool` | Указывает, авторизован ли текущий запрос к API. |
| **AuthWith** <br> `string` | Метод авторизации, если запрос был авторизован. |
| **Version** <br> `string` | Текущая версия API инстанса OpenVK. |
