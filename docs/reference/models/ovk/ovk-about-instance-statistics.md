# Объект OvkAboutInstanceStatistics

Представляет статистику инстанса OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **UsersCount** <br> `int` | Общее количество зарегистрированных пользователей на инстансе. |
| **OnlineUsersCount** <br> `int` | Количество пользователей, находящихся в данный момент в сети. |
| **ActiveUsersCount** <br> `int` | Количество активных пользователей на инстансе. |
| **GroupsCount** <br> `int` | Общее количество созданных сообществ. |
| **WallPostsCount** <br> `int` | Общее количество постов на стенах пользователей и сообществ. |

</div>
