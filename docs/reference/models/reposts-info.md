# Объект RepostsInfo (Информация о репостах ресурса)

Содержит краткие метаданные о том, сколько пользователей поделились данным ресурсом.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Count** <br> `int?` | Общее количество репостов данного объекта. |
| **UserReposted** <br> `bool?` | Поделился ли текущий авторизованный пользователь этим объектом. |

</div>
