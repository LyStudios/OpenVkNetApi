# Объект LikesInfo (Счетчик отметок ресурса)

Представляет собой информацию об отметках "Мне нравится" у любого комментируемого или оцениваемого объекта.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Count** <br> `int?` | Общее число пользователей, оценивших объект. |
| **UserLikes** <br> `bool?` | Поставил ли текущий пользователь отметку "Мне нравится". |
| **CanLike** <br> `bool?` | Может ли текущий пользователь поставить отметку "Мне нравится". |
| **CanPublish** <br> `bool?` | Может ли текущий пользователь поделиться (репостнуть) этим объектом. |

</div>
