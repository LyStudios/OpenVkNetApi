# Объект Post (Запись на стене)

Представляет собой полную запись на стене пользователя или сообщества в OpenVK API.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор записи. |
| **FromId** <br> `int` | Идентификатор автора записи. |
| **OwnerId** <br> `int` | Идентификатор владельца стены, на которой размещена запись. |
| **Date** <br> `long` | Unix-время публикации записи. |
| **PostType** <br> `string` | Тип записи (`post` — обычный пост, `copy` — репост, `reply` — ветка ответа). |
| **Text** <br> `string` | Текст публикации. |
| **CopyHistory** <br> `List<Post>` | История репостов (список предыдущих записей, если пост является репостом). |
| **Attachments** <br> `List<Attachment>` | Список вложений к записи (фотографии, видео, аудиозаписи, документы и др.). |
| **IsPinned** <br> `bool?` | Закреплена ли запись вверху стены. |
| **IsExplicit** <br> `bool?` | Помечена ли запись как контент для взрослых (NSFW). |
| **CanEdit** / **CanDelete** / **CanPin** / **CanArchive** <br> `bool?` | Флаги разрешений действий текущего пользователя с записью. |

</div>
