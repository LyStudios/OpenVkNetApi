# Объект AccountNameRequest (Заявка на смену имени)

Описывает текущую заявку на изменение имени/фамилии пользователя на инстансе.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор заявки. |
| **Status** <br> `string` | Статус заявки (например, `processing` — обрабатывается, `approved` — одобрена, `declined` — отклонена). |
| **FirstName** <br> `string` | Запрошенное новое имя. |
| **LastName** <br> `string` | Запрошенная новая фамилия. |

</div>
