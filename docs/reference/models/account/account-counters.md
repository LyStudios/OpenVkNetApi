# Объект AccountCounters (Счетчики аккаунта)

Представляет собой количество различных непрочитанных уведомлений, сообщений и запросов на аккаунте.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Friends** <br> `int?` | Количество входящих нерассмотренных запросов на добавление в друзья. |
| **Notifications** <br> `int?` | Количество новых непрочитанных системных уведомлений. |
| **Messages** <br> `int?` | Количество непрочитанных диалогов/личных сообщений. |

</div>
