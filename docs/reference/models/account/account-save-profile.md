# Объект AccountSaveProfile (Результат сохранения профиля)

Представляет собой результат сохранения изменений в профиле пользователя.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Changed** <br> `bool` | Указывает, были ли применены изменения в профиле. |
| **NameRequest** <br> [AccountNameRequest](/reference/models/account/account-name-request) | Информация о созданной заявке на смену имени и фамилии (если смена требует одобрения). |

</div>
