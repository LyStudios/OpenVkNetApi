# Объект AccountOvkSettings (Специфичные настройки OpenVK)

Описывает специфичные настройки интерфейса и отображения контента в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **AvatarStyle** <br> `int` | Стиль отображения аватаров (например, `0` — квадратные, `1` — круглые). |
| **Style** <br> `string` | Текущая выбранная тема или стиль оформления (например, `classic`, `dark`). |
| **ShowRating** <br> `bool` | Отображать ли рейтинг пользователя. |
| **NsfwTolerance** <br> `int` | Настройки фильтрации NSFW-контента (например, `0` — скрывать, `1` — предупреждать, `2` — показывать). |
| **PostView** <br> `string` | Режим отображения постов (например, классический или микроблог). |
| **MainPage** <br> `string` | Главная страница пользователя по умолчанию (например, `news` — новости, `profile` — личный профиль). |
