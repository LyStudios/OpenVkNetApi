# Объект AccountInfo (Настройки аккаунта)

Содержит сведения о настройках и состоянии аккаунта текущего пользователя.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **TwoFaRequired** <br> `int` | Указывает, требуется ли двухфакторная аутентификация для аккаунта. |
| **Country** <br> `string` | Двухбуквенный код страны пользователя. |
| **EuUser** <br> `bool` | Находится ли пользователь на территории Европейского союза. |
| **HttpsRequired** <br> `int` | Требуется ли использование HTTPS для запросов. |
| **CommunityComments** <br> `bool` | Включены ли комментарии от имени сообществ. |
| **IsLiveStreamingEnabled** <br> `bool` | Доступна ли функция прямых трансляций. |
| **Lang** <br> `int` | Числовой идентификатор языка интерфейса пользователя. |
| **NoWallReplies** <br> `int` | Отключены ли ответы на стене пользователя. |

</div>
