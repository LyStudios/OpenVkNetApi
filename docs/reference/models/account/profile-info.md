# Объект ProfileInfo (Информация профиля)

Содержит подробную личную информацию о профиле авторизованного пользователя.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **IsServiceAccount** <br> `bool?` | Является ли данный аккаунт сервисным/техническим. |
| **IsVerified** <br> `bool?` | Верифицирован ли данный профиль. |
| **VerificationStatus** <br> `string` | Статус или сообщение верификации профиля. |
| **HomeTown** <br> `string` | Родной город пользователя. |
| **Phone** <br> `string` | Номер телефона пользователя. |
| **Relation** <br> `int?` | Семейное положение (числовой код от 0 до 8). |
| **AudioStatus** <br> [Audio](/reference/models/audio/audio) | Аудиозапись, транслируемая в статус (если есть). |

</div>
