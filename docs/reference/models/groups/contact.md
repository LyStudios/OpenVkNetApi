# Объект Contact (Контактное лицо сообщества)

Представляет собой контактное лицо в списке контактов сообщества.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **UserId** <br> `int` | Идентификатор пользователя (контакта). |
| **Desc** <br> `string` | Описание должности или роли контакта (например, `"Администратор"`, `"По вопросам рекламы"`). |

</div>
