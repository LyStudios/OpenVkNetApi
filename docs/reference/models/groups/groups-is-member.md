# Объект GroupsIsMember (Членство в сообществе)

Содержит расширенную информацию о членстве и правах текущего пользователя в сообществе.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Member** <br> `bool` | Является ли текущий пользователь участником данного сообщества. |
| **Request** <br> `bool` | Подана ли незакрытая заявка на вступление в закрытое сообщество. |
| **Invitation** <br> `bool` | Имеется ли у пользователя неотклонённое приглашение в сообщество. |
| **CanInvite** <br> `bool` | Может ли текущий пользователь приглашать других в данное сообщество. |
| **CanRecall** <br> `bool` | Может ли текущий пользователь отозвать своё приглашение. |

</div>
