# Объект Group (Сообщество)

Представляет собой профиль сообщества (группы, публичной страницы или мероприятия) в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `long` | Уникальный числовой идентификатор сообщества. |
| **Name** <br> `string` | Название сообщества. |
| **ScreenName** <br> `string` | Короткое имя страницы (например, `club1` или буквенный псевдоним). |
| **Type** <br> `string` | Тип сообщества (`group` — группа, `page` — публичная страница, `event` — мероприятие). |
| **Description** <br> `string` | Текст описания сообщества. |
| **Site** <br> `string` | Ссылка на веб-сайт сообщества. |
| **MembersCount** <br> `int?` | Количество участников сообщества. |
| **IsMember** <br> `bool?` | Является ли текущий пользователь участником этого сообщества. |
| **Verified** <br> `bool?` | Подтвержден ли статус сообщества (галочка верификации). |
| **Photo50** / **Photo100** / **Photo200** <br> `string` | Ссылки на обложку (аватар) сообщества в различных разрешениях. |
| **CanPost** <br> `bool?` | Может ли текущий пользователь создавать записи на стене сообщества. |
| **RealId** <br> `int?` | Реальный отрицательный идентификатор сообщества (используется при работе со стеной). |

</div>
