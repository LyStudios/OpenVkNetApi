# Объект GroupBase (Базовая информация о сообществе)

Базовая модель сообщества, содержащая минимальный набор полей идентификации. Является родительским классом для полного объекта [Group](/reference/models/groups/group).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный числовой идентификатор сообщества. |
| **Name** <br> `string` | Название сообщества. |
| **ScreenName** <br> `string` | Короткое буквенное имя (например, `club1` или `mypublic`). |
| **IsClosed** <br> `bool?` | Является ли сообщество закрытым (приватным). |
| **CanAccessClosed** <br> `bool?` | Может ли текущий пользователь просматривать контент закрытого сообщества. |

</div>
