# Объект GroupSettings (Настройки сообщества)

Представляет собой конфигурацию настроек разделов и информации сообщества.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Title** <br> `string` | Название сообщества. |
| **Description** <br> `string` | Описание сообщества. |
| **Address** <br> `string` | Короткий адрес (короткое имя) сообщества. |
| **Website** <br> `string` | Адрес веб-сайта сообщества. |
| **Wall** <br> `int` | Настройки стены (`0` — отключена, `1` — открытая, `2` — ограниченная, `3` — закрытая). |
| **Photos** / **Video** / **Audio** / **Docs** / **Topics** <br> `int` | Уровень доступа к соответствующим разделам сообщества. |

</div>
