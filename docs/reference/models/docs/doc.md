# Объект Doc (Документ)

Представляет собой загруженный файл (документ) в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор документа. |
| **OwnerId** <br> `int` | Идентификатор владельца документа. |
| **Title** <br> `string` | Название файла документа. |
| **Size** <br> `long` | Размер файла в байтах. |
| **Extension** <br> `string` | Расширение файла (например, `txt`, `zip`, `pdf`). |
| **Url** <br> `string` | Прямая ссылка для скачивания файла. |
| **Date** <br> `long` | Unix-время загрузки документа. |
| **Type** <br> `int` | Числовой тип документа (текст, архив, изображение и т.д.). |
| **AccessKey** <br> `string` | Ключ доступа для приватных документов. |
| **Tags** <br> `List<string>` | Список тегов, присвоенных документу. |

</div>
