# Объект DocType (Тип документа)

Представляет собой информацию о типе документа и количестве файлов этого типа.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Type** <br> `int` | Идентификатор типа документа. |
| **Name** <br> `string` | Название типа документа (например, `"Книги"`, `"Архивы"`). |
| **Count** <br> `int` | Количество документов этого типа у пользователя. |

</div>
