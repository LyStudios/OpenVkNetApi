# Объект DocPreview (Предпросмотр документа)

Содержит данные для предварительного просмотра документа (изображения или видеозаписи).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Photo** <br> `DocPreviewPhoto` | Данные предпросмотра в виде фото (список размеров `PhotoSize`). |
| **Video** <br> `DocPreviewVideo` | Данные предпросмотра в виде видео (источник `src`, ширина, высота, размер файла). |

</div>
