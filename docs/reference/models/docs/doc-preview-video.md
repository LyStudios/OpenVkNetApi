# Объект DocPreviewVideo (Видео-превью документа)

Содержит информацию о видеозаписи, используемой для предварительного просмотра документа (например, анимации).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Src** <br> `string` | URL-ссылка на видеофайл превью. |
| **Width** <br> `int` | Ширина видео в пикселях. |
| **Height** <br> `int` | Высота видео в пикселях. |
| **FileSize** <br> `long` | Размер видеофайла в байтах. |

</div>
