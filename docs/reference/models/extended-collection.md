# Объект ExtendedCollection (Расширенная коллекция)

Коллекция элементов, содержащая списки дополнительных профилей пользователей и сообществ для авторов элементов. Наследует свойства [Collection](/reference/models/collection).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Profiles** <br> `List<User>` | Профили пользователей, связанных с элементами коллекции. |
| **Groups** <br> `List<Group>` | Профили сообществ, связанных с элементами коллекции. |
