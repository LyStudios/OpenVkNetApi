# Объект WallNearbyPost (Запись поблизости)

Представляет собой запись с географической меткой, опубликованную поблизости от указанных координат.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Message** <br> `string` | Текстовое содержимое записи. |
| **Url** <br> `string` | Ссылка на оригинальный пост на инстансе. |
| **Created** <br> `string` | Дата создания записи (в виде HTML-строки). |
| **Owner** <br> [WallNearbyPostOwner](/reference/models/wall/wall-nearby-post-owner) | Информация об авторе/владельце записи. |
| **Geo** <br> `Geo` | Географические координаты записи. |
| **Distance** <br> `float` | Расстояние до места публикации поста в километрах. |

</div>
