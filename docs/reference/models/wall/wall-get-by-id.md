# Объект WallGetById (Результат получения записей по ID)

Содержит информацию о записях стены, найденных по их уникальным идентификаторам.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Items** <br> `List<Post>` | Список найденных записей на стене [Post](/reference/models/wall/post). |
| **Profiles** <br> `List<ProfileInfo>` | Профили пользователей, связанных с данными записями. |
| **Groups** <br> `List<Group>` | Профили сообществ, связанных с данными записями. |
