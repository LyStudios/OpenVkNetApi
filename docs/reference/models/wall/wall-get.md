# Объект WallGet (Результат получения записей со стены)

Содержит список записей со стены, а также связанные профили пользователей и сообществ.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Count** <br> `int` | Общее количество записей на стене владельца. |
| **Items** <br> `List<Post>` | Список записей на стене [Post](/reference/models/wall/post). |
| **Profiles** <br> `List<ProfileInfo>` | Профили авторов записей (пользователей). |
| **Groups** <br> `List<Group>` | Профили сообществ-авторов записей. |
| **NextFrom** <br> `string` | Временной указатель для пагинации к следующей странице результатов. |
