# Объект WallNearbyPostOwner (Владелец записи поблизости)

Содержит краткую информацию о пользователе или сообществе, создавшем географическую запись поблизости.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Name** <br> `string` | Имя пользователя или сообщества. |
| **Domain** <br> `string` | Короткий буквенный адрес (домен) страницы владельца. |
| **Photo50** <br> `string` | URL-ссылка на 50px аватар владельца. |
| **Verified** <br> `bool` | Верифицирована ли страница автора. |

</div>
