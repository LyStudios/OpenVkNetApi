# Объект WallRepost (Результат репоста)

Содержит сведения о результате выполнения операции репоста объекта на стену.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Success** <br> `bool` | Указывает, была ли операция выполнена успешно. |
| **PostId** <br> `int` | Идентификатор новой созданной записи-репоста на стене. |
| **PrettyId** <br> `string` | Составной строковый идентификатор записи (в формате `owner_id_post_id`). |
| **RepostsCount** <br> `int` | Общее количество репостов для оригинального объекта с учётом нового. |
| **LikesCount** <br> `int` | Общее количество отметок "Мне нравится" у оригинального объекта. |

</div>
