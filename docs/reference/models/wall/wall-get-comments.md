# Объект WallGetComments (Результат получения списка комментариев)

Содержит список комментариев к записи на стене с метаданными ветки и правами доступа.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Count** <br> `int` | Общее число комментариев к записи. |
| **Items** <br> `List<WallComment>` | Список комментариев первого уровня [WallComment](/reference/models/comments/wall-comment). |
| **CurrentLevelCount** <br> `int` | Число комментариев на текущем уровне вложенности. |
| **CanPost** <br> `bool` | Может ли текущий пользователь комментировать эту запись. |
| **ShowReplyButton** <br> `bool` | Нужно ли отображать кнопку ответа. |
| **GroupsCanPost** <br> `bool` | Могут ли сообщества оставлять комментарии к записи. |
| **Profiles** <br> `List<ProfileInfo>` | Профили пользователей-авторов комментариев. |

</div>
