# Объект Post (Запись на стене)

Представляет собой запись на стене пользователя или сообщества в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор записи. |
| **FromId** <br> `int` | Идентификатор автора записи. |
| **OwnerId** <br> `int` | Идентификатор владельца стены, на которой опубликована запись. |
| **Date** <br> `long` | Unix-время публикации записи. |
| **PostType** <br> `string` | Тип записи (например, `"post"`, `"copy"` — репост, `"reply"` — ответ). |
| **Text** <br> `string` | Текстовое содержимое записи. |
| **CopyHistory** <br> `List<Post>` | История репостов (список предыдущих записей, если текущая является репостом). |
| **Attachments** <br> `List<Attachment>` | Список прикрепленных объектов (фото, видео, документы и др.). |
| **IsPinned** <br> `bool?` | Закреплена ли запись вверху стены. |
| **IsExplicit** <br> `bool?` | Содержит ли запись откровенный/взрослый контент (NSFW). |
| **CanEdit** / **CanDelete** / **CanPin** / **CanArchive** <br> `bool?` | Права доступа текущего авторизованного пользователя к записи. |
