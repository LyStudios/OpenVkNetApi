# Объект Poll (Опрос)

Представляет собой опрос в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор опроса. |
| **OwnerId** <br> `int` | Идентификатор создателя/владельца опроса. |
| **AuthorId** <br> `int` | Идентификатор автора опроса. |
| **Created** <br> `long` | Unix-время создания опроса. |
| **EndDate** <br> `long` | Unix-время окончания опроса (0, если бессрочный). |
| **Closed** <br> `bool` | Закрыт ли опрос для голосования. |
| **Multiple** <br> `bool` | Разрешен ли выбор нескольких вариантов ответа. |
| **Anonymous** <br> `bool` | Является ли опрос анонимным. |
| **Question** <br> `string` | Текст вопроса опроса. |
| **Votes** <br> `int` | Общее количество голосов. |
| **Answers** <br> `List<PollAnswer>` | Список возможных вариантов ответа (объекты `PollAnswer`). |
| **AnswerIds** <br> `List<int>` | Идентификаторы ответов, выбранных текущим пользователем. |
| **CanEdit** / **CanVote** / **CanShare** / **CanReport** <br> `bool` | Флаги разрешений для текущего авторизованного пользователя. |

</div>
