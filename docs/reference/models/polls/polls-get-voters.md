# Объект PollsGetVoters (Список проголосовавших)

Представляет собой список пользователей, проголосовавших за конкретный вариант ответа в опросе.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **AnswerId** <br> `int` | Идентификатор варианта ответа. |
| **Users** <br> `Collection<User>` | Коллекция пользователей [User](/reference/models/users/user), проголосовавших за данный вариант. |

</div>
