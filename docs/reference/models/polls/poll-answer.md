# Объект PollAnswer (Вариант ответа в опросе)

Представляет собой один из вариантов ответа в опросе со статистикой голосования.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `int` | Уникальный идентификатор варианта ответа. |
| **Text** <br> `string` | Текст варианта ответа. |
| **Votes** <br> `int` | Количество пользователей, выбравших этот вариант ответа. |
| **Rate** <br> `float` | Процент голосов, отданных за данный вариант. |

</div>
