# Объект PollsGetById (Расширенная информация об опросе)

Представляет собой расширенный ответ метода `polls.getById`, содержащий информацию об опросе и связанные профили пользователей. Наследует свойства [Poll](/reference/models/polls/poll).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Profiles** <br> `List<User>` | Профили пользователей, если метод был вызван с флагом `extended`. |
