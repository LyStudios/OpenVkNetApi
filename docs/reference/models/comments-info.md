# Объект CommentsInfo (Информация о комментариях ресурса)

Содержит краткую информацию о комментариях к объекту (записи на стене, видеозаписи и т.д.).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Count** <br> `int?` | Общее количество комментариев. |
| **CanPost** <br> `bool?` | Может ли текущий пользователь оставлять комментарии к этому объекту. |

</div>
