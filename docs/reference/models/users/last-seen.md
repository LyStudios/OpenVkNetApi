# Объект LastSeen (Последний визит)

Описывает время последнего визита пользователя и платформу, с которой он заходил.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Time** <br> `long?` | Время последнего визита в формате Unix-time. |
| **Platform** <br> `int?` | Идентификатор платформы (например, мобильная версия, приложение, браузер). |

</div>
