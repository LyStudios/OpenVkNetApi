# Объект Counters (Счетчики профиля)

Содержит количество различных объектов в профиле пользователя (друзья, фото, видео и т.д.).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Friends** <br> `int?` | Общее количество друзей пользователя. |
| **Photos** <br> `int?` | Общее количество загруженных фотографий. |
| **Videos** <br> `int?` | Общее количество загруженных видеозаписей. |
| **Audios** <br> `int?` | Общее количество добавленных аудиозаписей. |
| **Notes** <br> `int?` | Количество созданных заметок. |
| **Groups** <br> `int?` | Количество сообществ, в которых состоит пользователь. |
| **OnlineFriends** <br> `int?` | Количество друзей пользователя, находящихся в сети (online). |

</div>
