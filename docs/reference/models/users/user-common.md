# Объект UserCommon (Общая информация о пользователе)

Представляет собой общую информацию о профиле пользователя. Наследует свойства [UserBase](/reference/models/users/user-base).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Nickname** <br> `string` | Никнейм/отчество пользователя (если задано). |
| **ScreenName** <br> `string` | Буквенный псевдоним (короткое имя) страницы (например, `id1` или `username`). |
| **Sex** <br> `int?` | Пол пользователя (`1` — женский, `2` — мужской, `0` — не указан). |
| **Bdate** <br> `string` | Дата рождения (в формате `D.M.YYYY` или `D.M`). |
| **BdateVisibility** <br> `int?` | Видимость даты рождения в профиле. |
| **Status** <br> `string` | Текстовый статус профиля. |
| **Photo** <br> `string` | URL-адрес главной фотографии профиля. |
| **Photo50** / **Photo100** / **Photo200** <br> `string` | URL-адреса фотографий профиля в размерах 50x50, 100x100 и 200x200 пикселей. |
| **PhotoMediumRec** <br> `string` | URL-адрес фотографии среднего размера. |

</div>
