# Объект CustomField (Дополнительное поле)

Представляет собой значение настраиваемого дополнительного поля в профиле пользователя.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Type** <br> `int?` | Идентификатор типа поля. |
| **Name** <br> `string` | Название (заголовок) поля. |
| **Text** <br> `string` | Текстовое значение поля. |
