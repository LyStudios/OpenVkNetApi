# Объект UserBase (Базовая информация о пользователе)

Содержит минимальный набор полей для идентификации пользователя (ID, имя, фамилия). Является базовым классом для остальных моделей пользователя.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `long` | Уникальный числовой идентификатор пользователя. |
| **FirstName** <br> `string` | Имя пользователя. |
| **LastName** <br> `string` | Фамилия пользователя. |

</div>
