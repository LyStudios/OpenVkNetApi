# Объект User (Пользователь)

Представляет собой полный профиль пользователя в OpenVK.

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Id** <br> `long` | Уникальный идентификатор пользователя. |
| **FirstName** <br> `string` | Имя пользователя. |
| **LastName** <br> `string` | Фамилия пользователя. |
| **Nickname** <br> `string` | Никнейм пользователя. |
| **ScreenName** <br> `string` | Короткое имя страницы (например, `id1` или буквенный псевдоним). |
| **Sex** <br> `int?` | Пол (`1` — женский, `2` — мужской, `0` — не указан). |
| **Bdate** <br> `string` | Дата рождения (в формате `D.M.YYYY` или `D.M`). |
| **Status** <br> `string` | Текстовый статус профиля. |
| **Photo50** / **Photo100** / **Photo200** / **Photo** <br> `string` | Ссылки на аватар пользователя в разном разрешении. |
| **Online** <br> `bool?` | Находится ли пользователь в данный момент в сети. |
| **Verified** <br> `bool?` | Подтвержден ли профиль галочкой верификации. |
| **Deactivated** <br> `string` | Статус деактивации аккаунта (например, `"deleted"` или `"banned"`). |
| **IsClosed** <br> `bool?` | Закрыт ли профиль настройками приватности. |
| **CanAccessClosed** <br> `bool?` | Доступен ли закрытый профиль для просмотра текущим пользователем. |
| **About** / **Interests** / **Music** / **Movies** / **Tv** / **Books** / **Games** / **Quotes** <br> `string` | Различные текстовые разделы о себе и своих интересах. |

</div>
