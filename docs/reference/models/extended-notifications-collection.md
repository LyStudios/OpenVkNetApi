# Объект ExtendedNotificationsCollection (Расширенная коллекция уведомлений)

Коллекция уведомлений, дополненная профилями пользователей и сообществ. Наследует свойства [NotificationsCollection](/reference/models/notifications-collection).

</div>

<div class="vk-card">

# Поля объекта

| Поле / Тип | Описание |
| :--- | :--- |
| **Profiles** <br> `List<ProfileInfo>` | Профили пользователей, вызвавших появление уведомлений. |
| **Groups** <br> `List<Group>` | Профили сообществ, связанных с уведомлениями. |
