# friends.delete

Удаляет пользователя из списка друзей или отклоняет/удаляет входящую или исходящую заявку.

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя.</span>
</div>

# Вызов метода

```csharp
int result = await api.Friends.DeleteAsync(userId: "1");
```

</div>

<div class="vk-card">

# Параметры

| Параметр / Тип | Описание |
| :--- | :--- |
| **userId** <br> `string` | Идентификатор пользователя, которого нужно удалить из друзей/заявок. <br> <span style="color: var(--vp-c-text-3)">строка, обязательный параметр</span> |

</div>

<div class="vk-card">

# Результат

Возвращает числовой код результата выполнения операции:
* `1` — пользователь успешно удален из друзей;
* `2` — отклонен входящий запрос на добавление;
* `3` — отменен исходящий запрос.

</div>
