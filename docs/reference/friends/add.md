# friends.add

Одобрение входящей заявки или отправка исходящего запроса на добавление в друзья.

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя.</span>
</div>

# Вызов метода

```csharp
int result = await api.Friends.AddAsync(userId: "1");
```

</div>

<div class="vk-card">

# Параметры

| Параметр / Тип | Описание |
| :--- | :--- |
| **userId** <br> `string` | Идентификатор пользователя, которого нужно добавить в друзья. <br> <span style="color: var(--vp-c-text-3)">строка, обязательный параметр</span> |

</div>

<div class="vk-card">

# Результат

Возвращает числовой код результата выполнения операции:
* `1` — отправлен запрос на добавление в друзья;
* `2` — входящая заявка от этого пользователя одобрена (пользователи стали друзьями).

</div>
