# groups.join

Вступает в сообщество (группу, публичную страницу, мероприятие).

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя. Для закрытых групп метод отправляет заявку на вступление.</span>
</div>

# Вызов метода

```csharp
int result = await api.Groups.JoinAsync(groupId: 12);
```

</div>

<div class="vk-card">

# Параметры

| Параметр / Тип | Описание |
| :--- | :--- |
| **groupId** <br> `int` | Идентификатор сообщества, в которое нужно вступить. <br> <span style="color: var(--vp-c-text-3)">целое число, обязательный параметр</span> |

</div>

<div class="vk-card">

# Результат

Возвращает `1` в случае успешного вступления или отправки заявки.

</div>
