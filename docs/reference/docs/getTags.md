# docs.getTags

Возвращает список тегов документов пользователя или сообщества.

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя.</span>
</div>

# Вызов метода

```csharp
var tagsList = await api.Docs.GetTagsAsync(ownerId: 1, type: 0);
```

</div>

<div class="vk-card">

# Параметры

| Параметр / Тип | Описание |
| :--- | :--- |
| **ownerId** <br> `int?` | Идентификатор владельца документов (если не передан, используется ID текущего пользователя). <br> <span style="color: var(--vp-c-text-3)">целое число, необязательный параметр</span> |
| **type** <br> `int?` | Тип документов для фильтрации тегов. По умолчанию 0 — все типы. <br> <span style="color: var(--vp-c-text-3)">целое число, необязательный параметр</span> |

</div>

<div class="vk-card">

# Результат

Возвращает список строк `List<string>`, содержащий все найденные теги документов.

</div>
