# newsfeed.addBan

Добавляет указанных пользователей или сообщества в черный список ленты новостей (скрывает их публикации из ленты текущего пользователя).

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя.</span>
</div>

# Вызов метода

```csharp
int result = await api.Newsfeed.AddBanAsync(userIds: "1,2", groupIds: "10,20");
```

</div>

<div class="vk-card">

# Параметры

| Параметр / Тип | Описание |
| :--- | :--- |
| **userIds** <br> `string` | Идентификаторы скрываемых пользователей, разделенные запятыми. <br> <span style="color: var(--vp-c-text-3)">строка, необязательный параметр</span> |
| **groupIds** <br> `string` | Идентификаторы скрываемых сообществ, разделенные запятыми. <br> <span style="color: var(--vp-c-text-3)">строка, необязательный параметр</span> |

</div>

<div class="vk-card">

# Результат

Возвращает `1` в случае успешного добавления в черный список ленты.

</div>
