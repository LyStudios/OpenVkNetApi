# newsfeed.deleteBan

Удаляет указанных пользователей или сообщества из черного списка ленты новостей (публикации снова начинают отображаться в ленте).

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя.</span>
</div>

# Вызов метода

```csharp
int result = await api.Newsfeed.DeleteBanAsync(userIds: "1,2", groupIds: "10,20");
```

</div>

<div class="vk-card">

# Параметры

| Параметр / Тип | Описание |
| :--- | :--- |
| **userIds** <br> `string` | Идентификаторы пользователей, которых нужно вернуть в ленту, разделенные запятыми. <br> <span style="color: var(--vp-c-text-3)">строка, необязательный параметр</span> |
| **groupIds** <br> `string` | Идентификаторы сообществ, которые нужно вернуть в ленту, разделенные запятыми. <br> <span style="color: var(--vp-c-text-3)">строка, необязательный параметр</span> |

</div>

<div class="vk-card">

# Результат

Возвращает `1` в случае успешного удаления из черного списка ленты.

</div>
