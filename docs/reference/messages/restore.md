# messages.restore

Восстанавливает ранее удаленное личное сообщение.

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя. Восстановление возможно только в том случае, если сообщение еще не было полностью стерто из базы данных.</span>
</div>

# Вызов метода

```csharp
int result = await api.Messages.RestoreAsync(messageId: 12);
```

</div>

<div class="vk-card">

# Параметры

| Параметр / Тип | Описание |
| :--- | :--- |
| **messageId** <br> `int` | Идентификатор восстанавливаемого сообщения. <br> <span style="color: var(--vp-c-text-3)">целое число, обязательный параметр</span> |

</div>

<div class="vk-card">

# Результат

Возвращает `1` в случае успешного восстановления.

</div>
