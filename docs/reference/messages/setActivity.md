# messages.setActivity

Отправляет статус активности (например, информацию о том, что текущий пользователь набирает текст) в указанный диалог.

<div class="vk-warning">
  <span>💡 Для вызова этого метода требуется авторизация пользователя. Статус отображается собеседнику в течение 10 секунд или до отправки нового сообщения.</span>
</div>

# Вызов метода

```csharp
int result = await api.Messages.SetActivityAsync(type: "typing", peerId: 12);
```

</div>

<div class="vk-card">

# Параметры

| Параметр / Тип | Описание |
| :--- | :--- |
| **peerId** <br> `int` | Идентификатор диалога/собеседника, которому отправляется статус. <br> <span style="color: var(--vp-c-text-3)">целое число, обязательный параметр</span> |
| **type** <br> `string` | Тип активности. На данный момент поддерживается только `typing` (набор текста). <br> <span style="color: var(--vp-c-text-3)">строка, по умолчанию: "typing"</span> |

</div>

<div class="vk-card">

# Результат

Возвращает `1` в случае успешного выполнения.

</div>
